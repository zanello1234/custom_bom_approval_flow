üìã How It Works

  1. Mark Product as Flexible: Enable "Flexible BOM" checkbox on products that need custom configuration.
  2. Create Base BOM: Set up a template BOM with standard components and operations.
  3. Configure in Sales Order: Click "Configure BOM" button when adding flexible products to sales orders.
  4. Customize Components: Use the wizard to add, remove, or modify components and quantities.
  5. Choose BOM Type: Select between Manufacturing BOM or Kit based on customer needs.
  6. Generate Custom BOM: System creates a quote-specific BOM linked to the sales order line.
  7. Production Integration: Manufacturing orders automatically use the custom BOM configuration.
